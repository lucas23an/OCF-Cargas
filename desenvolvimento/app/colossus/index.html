Como Funciona


O Colossus é um Framework que auxilia o programador BackEnd no desenvolvimento de qualquer projeto, atravez dele o programador irá ganhar mais tempo no desenvolvimento do projeto, ele da a base "Molde" do código fonte, ex: View do painel, Controller, Model e Conexão com o Banco.

Possibilitando assim gastar uma hora para fazer o mesmo que levaria um mês. Para utilizar é bem simples basta ter em mãos o nome do banco de dados, as tabelas e as colunas. Atravez do Colossus você terá o CRUD de cada tabela do seu banco de dados sql.
